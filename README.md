# Decathlon Schedule

Ce projet propose une alternative simple pour consulter le planning des cours.
Il n’a pas vocation a remplacer l’existant et reste complementaire.

Ce site est realise benevolement, comme une alternative pour faciliter le suivi du planning.

Lien : https://decathlon-schedule.vercel.app/

## Installation

- `npm install`
- `npm run dev`

## Build

- `npm run build`
- `npm run preview`
